### Description
- This entry serves as the reference implementation for Anthropic's official agentic coding tool featuring a robust permission model with per-tool prompts. The system utilizes a 1M token context window through the Opus 4.7 model to handle massive codebases with high reasoning accuracy. Multi-surface continuity ensures the same engine powers the terminal, desktop, and web environments for a seamless development experience. Security is maintained via project-scoped allowlists that restrict agent actions to specific directories and tools.
