# OpenCode - https://opencode.ai

OpenCode is a provider-agnostic, open-source alternative to Claude Code with a heavy focus on the terminal user interface (TUI). It is developed by the creators of terminal.shop with the intent to maximize developer productivity within terminal environments.

### Description
- OpenCode utilizes a distinct agent architecture featuring a "build" agent for active development and a "plan" agent for read-only analysis. The tool is designed with a client/server architecture, allowing the TUI to run as a remote client while the server handles core logic. It includes out-of-the-box support for Language Server Protocol (LSP) to enhance code intelligence during sessions. The project maintains a philosophy of being 100% open source and decoupled from any specific model provider.
