### Description
- OpenClaw utilizes a local-first gateway as a single control plane for managing sessions, tools, and events. Originally inspired by the "Molty" space lobster project, it supports over 20 messaging channels including WhatsApp, Telegram, and iMessage. The security model defaults to DM pairing, requiring unknown senders to provide a pairing code before the bot processes their input. It also features a live Canvas for visual work and supports companion apps for macOS and mobile operating systems. The system is designed to be highly versatile, offering voice wake capabilities and multi-agent routing for isolated tasks.
