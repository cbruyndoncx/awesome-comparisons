### Description
- Goose is built using Rust to ensure high performance and portability across multiple operating systems. The project recently migrated from Block to the Linux Foundation’s Agentic AI Foundation to foster a more open governance model. It supports more than 15 model providers, allowing users to leverage their existing subscriptions through various authentication methods. The tool is extensible via the Model Context Protocol, providing access to dozens of pre-configured extensions.
