# Gemini CLI - https://github.com/google-gemini/gemini-cli

Gemini CLI is a terminal-first agent providing an open-source path to interact directly with Google’s Gemini model family. It is optimized for developers requiring high-reasoning capabilities and extensive automation within the command line.

### Description
- The tool offers a generous free tier for personal Google accounts and supports advanced reasoning via Gemini 3 models. It includes grounding with Google Search for real-time information retrieval during coding tasks. Integration extends to GitHub Actions, enabling automated pull request reviews and issue triage directly in the repository. Gemini CLI also features multimodal capabilities, allowing it to generate apps from PDFs or sketches. Users can also leverage token caching to optimize their usage during long sessions.
