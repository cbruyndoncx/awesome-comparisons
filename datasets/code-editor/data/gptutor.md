### Description
GPTutor is a Visual Studio Code extension launched in 2025 that specializes in explaining code and helping developers learn. Unlike code generation tools, GPTutor focuses on understanding existing code, providing detailed explanations of how code works, why certain patterns are used, and offering learning resources. It's particularly useful for developers working with unfamiliar codebases or learning new programming concepts and patterns.
