### Description

Replit is a cloud-based development environment that runs entirely in the browser. It combines a full IDE, AI coding assistance (Ghostwriter), autonomous agent (Replit Agent), deployment, and collaboration in a single platform. Ghostwriter provides real-time inline code completions, explanations, refactoring suggestions, and debugging assistance. Replit Agent autonomously plans projects, creates files, installs dependencies, and iterates toward a working application — capable of 200-minute autonomous runs in Agent 3 (2026).

The platform targets beginners, educators, rapid prototypers, and teams that want to avoid local environment setup. Deployment to Replit's infrastructure is one-click, with custom domain support and compute scaling. Collaboration is real-time with multiplayer editing up to 15 users on Pro tier.
