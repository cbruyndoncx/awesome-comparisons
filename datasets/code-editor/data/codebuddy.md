# CodeBuddy - https://codebuddy.tencent.com
CodeBuddy is a full-stack AI programming tool developed by Tencent Cloud, powered by Tencent Hunyuan and DeepSeek V3 dual language models. Available as both a standalone IDE and plugin for VS Code and JetBrains IDEs, CodeBuddy provides end-to-end development assistance with intelligent code generation, design-to-code conversion, and autonomous multi-file code modifications through its flagship Craft Mode feature.

### Short Description
Tencent's full-stack AI IDE with autonomous code generation and design-to-code capabilities

### Description
CodeBuddy operates as a comprehensive AI coding assistant available in three forms: standalone IDE, VS Code plugin, and JetBrains plugin. Over 90% of Tencent's engineers use CodeBuddy daily, with the tool generating 50% of the company's newly written code and achieving a 40% reduction in coding time.

The platform's Craft Mode enables developers to use natural language instructions to direct the AI in autonomously understanding requirements, generating multi-file code, and performing code modifications without manual file-by-file editing. CodeBuddy converts Figma designs to production-ready code with 99.9% accuracy, includes built-in Supabase integration for instant database and authentication, and provides one-click deployment for transitioning from development to live demo.
