### Description
Factory introduced "Droids" as autonomous software development agents that became generally available on May 28, 2025, after raising $50 million in Series B funding. Droids work across the entire software development lifecycle, independently building production-ready features from tickets, specs, or prompts. With a score of 58.75%, Droid sets the new state-of-the-art on Terminal-Bench, surpassing Claude Code with Opus (43.2%) and Codex CLI (42.8%). The platform offers specialized capabilities including autonomous end-to-end coding, incident resolution with root cause analysis, Knowledge Droid for codebase understanding and documentation generation, and automated Linear/Jira management. Droids can be used in any IDE (VS Code, JetBrains, Vim) or terminal, with native integrations for GitHub/GitLab, Jira, Slack, and PagerDuty. Enterprise customers include Ernst & Young, Nvidia, MongoDB, Zapier, Bayer, and Clari.
