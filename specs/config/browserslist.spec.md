### Defines supported browsers per Angular 17 defaults

Specifies the list of browsers to target during builds:

- `last 2 Chrome versions`
- `last 2 Firefox versions`
- `last 2 Edge versions`
- `last 2 Safari versions`
- `last 2 iOS versions`
- `Firefox ESR`
- excludes `IE 11`
